# Quentin Ikuta Multi Page Portfolio

This package is GitHub Pages ready and uses a simple static multi page structure.

## Included pages
- `index.html` — landing page only
- `about.html`
- `ifm.html`
- `featured.html`
- `speaking.html`
- `research.html`
- `updates.html`
- `styles.css`
- `assets/images/` — uploaded photos used throughout the site

## Deploy on GitHub Pages
1. Create or open your repository.
2. Upload all files and folders from this package to the repository root.
3. In GitHub go to **Settings → Pages**.
4. Set source to **Deploy from a branch**.
5. Choose your main branch and root folder.
6. Save.

## Notes
- The landing page is now just the intro.
- Each major content area has its own page.
- Uploaded photos were added into the design in multiple places.
- To update speaking items, edit the `engagements` array in `speaking.html`.
- To update the feed, edit the `updates` array in `updates.html`.
